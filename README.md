# CS301_Assignment_1
Я створила базу даних для кінотеатру, у якій є 5 таблиць: користувачі, фільми, кінотеатри, сеанси та квитки. Користувачі можуть купувати квитки, кожен фільм має свої сеанси, а кінотеатри показують різні фільми. Квитки пов’язані з конкретним користувачем і сеансом, що дозволяє відстежувати всі покупки та розклад показів.
